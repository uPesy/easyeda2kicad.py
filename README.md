# easyeda2kicad by Wadam100

A Python script that converts electronic components from [EasyEDA](https://easyeda.com/) or [LCSC](https://www.lcsc.com/) to KiCad libraries including **3D models** in color. This version includes **KiCad 10** output format support.

Speeds up your PCB design workflow, especially when using [JLCPCB SMT assembly services](https://jlcpcb.com/caa).

---

## What You Need

- **Python 3.6** or newer
- **pip** (comes with Python)
- **Git**

## How to Install

### Option 1 - One-line install (recommended)

```bash
pip install git+https://github.com/Wadam100/easyeda2kicad.py.git
```

To update later:

```bash
pip install --upgrade git+https://github.com/Wadam100/easyeda2kicad.py.git
```

### Option 2 - Download and install manually

1. Download the code:

```bash
git clone https://github.com/Wadam100/easyeda2kicad.py.git
cd easyeda2kicad.py
```

2. Install it:

```bash
pip install .
```

If you plan to edit the code yourself, use this instead:

```bash
pip install -e .
```

### Check that it worked

```bash
easyeda2kicad --help
```

You should see a list of available options.

## How to Use

Convert a component by its LCSC part number:

```bash
# Everything (symbol + footprint + 3D model)
easyeda2kicad --full --lcsc_id=C2040

# Pick what you need
easyeda2kicad --symbol --lcsc_id=C2040
easyeda2kicad --footprint --lcsc_id=C2040
easyeda2kicad --3d --lcsc_id=C2040

# Or combine them
easyeda2kicad --symbol --footprint --3d --lcsc_id=C2040
```

### Where files are saved

By default, libraries are saved to:
- **Windows:** `C:/Users/your_name/Documents/Kicad/easyeda2kicad/`
- **Linux:** `/home/your_name/Documents/Kicad/easyeda2kicad/`

Files created:
- `easyeda2kicad.kicad_sym` - symbol library
- `easyeda2kicad.pretty/` - footprint library
- `easyeda2kicad.3dshapes/` - 3D models (.WRL and .STEP)

### Save to a custom location

```bash
easyeda2kicad --full --lcsc_id=C2040 --output ~/libs/my_lib
```

### Overwrite an existing component

```bash
easyeda2kicad --full --lcsc_id=C2040 --output ~/libs/my_lib --overwrite
```

### KiCad v5 legacy format

```bash
easyeda2kicad --symbol --lcsc_id=C2040 --v5
```

## Setting Up Libraries in KiCad

Before configuring KiCad, run the script at least once to create the library files:

```bash
easyeda2kicad --full --lcsc_id=C2040
```

Then in KiCad:

1. Go to **Preferences > Configure Paths** and add a new variable called `EASYEDA2KICAD`:
   - **Windows:** `C:/Users/your_username/Documents/Kicad/easyeda2kicad/`
   - **Linux:** `/home/your_username/Documents/Kicad/easyeda2kicad/`
2. Go to **Preferences > Manage Symbol Libraries** and add: `${EASYEDA2KICAD}/easyeda2kicad.kicad_sym`
3. Go to **Preferences > Manage Footprint Libraries** and add: `${EASYEDA2KICAD}/easyeda2kicad.pretty`
